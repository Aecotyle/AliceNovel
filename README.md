# NEO_Engine

## 概要
こちらは[MAUI]を採用したテキスト系ゲームのUI部分となっています。<br>
ベースとしてこちらを採用することにより、UI部分の構築の手間が減り作成が楽になると思います。<br>

現時点では未完成で、開発中となっています。<br>
