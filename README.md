# Alice Novel

## 概要
こちらは[MAUI]を採用したテキスト系ゲームのUI部分となっています。<br />
ベースとしてこちらを採用することにより、UI部分の構築の手間が減り作成が楽になります。<br />

### 技術
プログラミング言語<br />

- [C#] + [xaml]

フレームワーク<br />

- [MAUI] / [.NET7.0]
